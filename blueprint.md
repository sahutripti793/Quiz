
# Mind Hustlers Quiz Blueprint

## Overview

A simple quiz application, now named "Mind Hustlers", that presents multiple-choice questions to the user, tracks their score, and displays the result at the end.

## Implemented Features

### Core Functionality

*   **Quiz Start:** A "Start Quiz" button to begin the quiz.
*   **Question Display:** Questions are shown one at a time.
*   **Multiple-Choice Questions:** The quiz consists of at least 5 MCQs with four options each.
*   **Answer Selection:** Users can select a single answer for each question.
*   **Navigation:** A "Next" button allows users to proceed to the following question.
*   **Score Calculation:** The final score is calculated based on the number of correct answers.
*   **Results Display:** The total score is presented to the user upon completion of the quiz.

### Design and UX

*   **Responsive Design:** The application is responsive and works on different screen sizes.
*   **User Feedback:** Visual cues for selected answers and quiz progression.
*   **Branding:** The application is named "Mind Hustlers".
*   **Animated Background:** A dynamic background with large, floating arithmetic operators, mathematical tools (ruler, calculator), and brain icons. The icons float freely without a visible container, creating a cleaner and more polished look.
*   **Colorful Icons:** The animated background icons have been given a vibrant and varied color palette, enhancing the visual appeal of the application.
*   **Prominent Brain Icons:** The brain icons in the background animation have been made larger to give them more prominence and reinforce the quiz theme.
*   **New Color Scheme:** The application now features a new color scheme with a beige background and an olive quiz container, giving it a more earthy and scholarly feel.
*   **Quiz Box Styling:** The quiz box is now more transparent, narrower, and taller for a more sleek and modern look.
